## Features

- Increased capacity of Paramedic Outfit (30 => 500, 16.67x)
- Increased capacity of Everyday/Sports Backpacks (240 => 320, 1.33x)

## Installation

To build the mod from source, install [HEMTT] and run either of these commands:

```sh
hemtt build
# or:
hemtt release
```

[HEMTT]: https://github.com/brettmayson/HEMTT
