# 420th Customizations

An Arma 3 mod for the 420th Delta community.

## Installation

To build the mod from source, install [HEMTT] and run either of these commands:

```sh
hemtt build
# or:
hemtt release
```

[HEMTT]: https://github.com/brettmayson/HEMTT
